About Us

Purpose

This website was created to help pre-service ICT teachers improve their ICT knowledge and skills. It provides online tutorial videos that help future teachers learn how to use interactive technology and websites for effective teaching.

Mission

Our mission is to help pre-service ICT teachers develop their digital skills through simple and helpful online tutorials.

Vision

To become a useful learning platform where future teachers can learn how to use technology confidently in teaching.

Description

The website provides online tutorial videos for pre-service teachers. These videos show step-by-step guides on how to use different websites for lesson preparation and classroom activities. The goal is to make learning ICT skills easy and accessible anytime.
